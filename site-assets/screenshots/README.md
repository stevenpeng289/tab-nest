## Capture Guidelines

- Use PNG format.
- Prefer desktop screenshots around `2560px` wide or larger.
- Keep browser UI out of the screenshot when possible.
- Use realistic tab/bookmark examples, but avoid exposing private data.
- Capture one focused feature per screenshot.
- Keep the visual theme consistent across the set.
