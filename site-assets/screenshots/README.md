# Screenshots

Put your GitHub Pages / Chrome Web Store product screenshots in this folder.

Recommended filenames for the current homepage:

- `homepage.png`
- `windowsPage.png`
- `opensession.png`

Recommended export tips:

- Use PNG format
- Prefer widths around `1280px` or higher
- Keep the screenshots clean and focused on one core feature each

The root `index.html` already references these filenames directly.
